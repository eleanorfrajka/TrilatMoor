trilatmoor

trilatmoor is a Python package for determining the seafloor position of an oceanographic mooring anchor from acoustic ranges measured at the sea surface during a trilateration survey.

The package implements the least-squares trilateration algorithm originally developed in MATLAB for the RAPID 26°N mooring array, which makes continuous estimates of the Atlantic Meridional Overturning Circulation.
